# Tennis Coffee Table

Односторінковий інтернет-магазин (SPA) дизайнерських кавових столиків, виготовлених із перероблених тенісних м’ячів та загартованого скла. Мінімалістичний «галерейний» дизайн у світлих тонах із природним жовто-зеленим акцентом.

> Це демо-проєкт: товари, ціни, фото (SVG-плейсхолдери) та тексти — заглушки.

## ✨ Можливості

- Хедер із навігацією (якір-посилання: Каталог, Про нас, Контакти) та мобільне меню.
- Головний банер (hero) зі слоганом та CTA-кнопкою.
- Каталог із 6 моделей — картки з фото, назвою, ціною та кнопкою замовлення.
- Секція «Про нас» — історія бренду та екологічність (переробка).
- Форма замовлення / контактів (демо, без відправки на сервер).
- Футер із контактами та соцмережами.
- Плавний скрол до секцій та анімація появи при скролі.
- Адаптивний дизайн (mobile-first).

## 🛠 Стек

- **React 18** + **Vite 5** — швидкий старт та миттєвий деплой на Vercel.
- Чистий CSS (без залежностей), CSS-змінні, `clamp()` для типографіки.
- Без зовнішніх зображень — всі «фото» є інлайн-SVG плейсхолдерами.

## 📁 Структура проєкту

```
tennis-coffee-table/
├─ public/
│  └─ favicon.svg        # іконка сайту
├─ src/
│  ├─ App.jsx           # усі секції SPA + дані товарів
│  ├─ main.jsx          # точка входу React
│  └─ index.css         # глобальні стилі та адаптив
├─ index.html            # HTML-оболонка
├─ vite.config.js        # конфіг Vite (плагін React)
├─ vercel.json           # SPA-rewrites для Vercel
├─ package.json
└─ README.md
```

## 🎨 Дизайн

- Кольорова гама: білий та світлі тони, світло-сірі акценти.
- Акцентний колір — природний жовто-зелений (`#c8e04f`), дозовано.
- Типографіка: системний sans-serif, тонкі лінії, багато повітря.

## 🚀 Запуск локально

Потрібно Node.js 18+.

```bash
npm install
npm run dev      # локальний сервер http://localhost:5173
npm run build    # збірка у dist/
npm run preview  # перегляд збірки
```

## ☁️ Деплой: GitHub + Vercel

### 1. Репозиторій GitHub

Код уже у цьому репозиторії. Якщо працюєте локально:

```bash
git clone https://github.com/aimykola/tennis-coffee-table.git
cd tennis-coffee-table
# внесіть зміни, потім:
git add .
git commit -m "update"
git push origin main
```

### 2. Деплой на Vercel (через інтерфейс)

1. Зайдіть на https://vercel.com та увійдіть через GitHub.
2. Натисніть **Add New → Project**.
3. Оберіть репозиторій `tennis-coffee-table` та натисніть **Import**.
4. Vercel автоматично визначить Vite:
   - **Framework Preset:** Vite
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
   - **Install Command:** `npm install`
5. Натисніть **Deploy** і зачекайте ~1 хвилину.
6. Отримаєте посилання вигляду `https://tennis-coffee-table.vercel.app`.

Після цього кожен `git push` у гілку `main` автоматично перезбирає та оновлює сайт.

### 3. Альтернатива: Vercel CLI

```bash
npm i -g vercel
vercel        # прев'ю-деплой
vercel --prod # продакшн
```

## 📝 Нотатки

- Файл `vercel.json` містить SPA-rewrites, щоб всі маршрути вели на `index.html`.
- Моделі товарів та тексти можна змінити у масиві `PRODUCTS` у файлі `src/App.jsx`.
